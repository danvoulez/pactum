# Documentation

This directory contains notes and specifications from the development of Pactum RiskPact V0.2.

## Files

- `00-spec-v0.md` - Initial specification for canonical JSON, hashing, and signing
- `01-v0.1-hardening.md` - V0.1 hardening notes (error codes, documentation)
- `02-v0.2-spec.md` - V0.2 specification (multi-round envelopes, gap detection)
- `03-v0.1-implementation.md` - V0.1 implementation milestones
- `04-milestone-notes.md` - Development milestone notes
- `05-v0-initial.md` - Initial V0 implementation notes
- `06-case3-4-5-code.md` - Code references for test cases 3-5
- `07-case6-code.md` - Code reference for test case 6

These files document the evolution of the protocol and implementation decisions made during development.
